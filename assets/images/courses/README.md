# Course cover placeholders (prototype)

Drop client-provided cover art here as **`{course-id}.jpg`** (16:9, e.g. 640×360 or larger).

Examples: `IND-1.jpg`, `NOR-3.jpg`, `MLL-9.jpg`

The prototype loads covers from this folder via `courseCoverUrl()` in `js/mi-formacion.js`. Replacing a file keeps the same filename — no code change required.

Current files are **temporary stock placeholders** only; replace before production or client demo.
